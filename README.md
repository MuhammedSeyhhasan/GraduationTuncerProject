#TUNCER OGLU PROJECT 

Web Project for a Company (PHP / Laravel + Admin Panel + Chatbot)
This is a web application developed for a wholesale company using PHP and Laravel.
It includes an Admin Panel to manage products and pages, and also features a Chatbot built with Dialogflow.


##Technologies Used

PHP / Laravel Framework
MySQL Database
JavaScript
Dialogflow ES (for chatbot)

##CHATBOT
You can find the exported Dialogflow agent in the folder: dialogflow-agent/
